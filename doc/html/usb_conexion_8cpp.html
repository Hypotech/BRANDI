int enviar_Dato_USB ( unsigned char *  ,

)

Manda datos al dispositivo USB con el que cuenta BRANDI.

Esta función comprueba que el dato no supere el numero de buffers con los que cuenta el dispositivo USB. También comprueba que no hubo error a la hora de enviar los Datos, en caso de error imprime en pantalla el error asociado.

Parámetros
[in]DatosApuntador a una cadena de Bytes (unsigned char).
[in]lengthLongitud de la cadena Datos (este valor nunca debe ser mayor que NUM_BUFFERS).
Devuelve
El número de bytes transferidos. -1 en caso de haber un error con el envio de los datos. -2 si la longitud supera a NUM_BUFFERS.
int inicializar_USB ( void  )

Configura la comunicación USB.

Busca el dispositivo USB, reclama el dispositivo al S.O. y crea una tuberia de comunicación. Esta función debe ser invocada antes de hacer cualquier operación de comunicación con BRANDI.

Devuelve
0 en caso de éxito, -1 si no se puede crear una sesión libusb, -2 si no se pudo abrir el Dispositivo USB, -3 en caso de que fallar la reclamación al S.O.
