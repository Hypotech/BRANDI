float Convierte_a_Angulo ( const coordenada2D  C1,
const coordenada2D  C2 
)

Función de utilidad para convertir coordenadas de Kinect en ángulos.

Esta función es utilizada por la función GetKinectFrame para determina el ángulo de una extremidad con respecto a otra, dadas sus coordenadas en el espacio Kinect.

Parámetros
[in]C1Coordenadas de la extremidad uno.
[in]C2Coordenadas de la extremidad dos.
Devuelve
El ángulo que existe entre la extremidad 1 y la extremidad 2.

int GetKinectFrame ( )

Función que obtiene los ángulos del Kinect.

Obtiene los ángulos de las muñecas izquierda y derecha; de las manos izquierda y derecha; y codos izquierdo y derecho. Los valores son almacenados en las variables globales: AnguloManoIzq, AnguloMunyecaIzq, AnguloCodoIzq, AnguloManoDer, AnguloMunyecaDer, AnguloCodoDer.

Devuelve
1 en caso de exito, 0 en caso de que no se tenga ningún esqueleto.

void ini_Kinect ( void  )

Configura el kinect de manera adecuada para su utilización.

Esta función debe llamarse antes de cualquier operación con el Kinect.Esta función le indica al Kinect su uso en modo esqueleto. Una vez invocada esta función sólo dará importancia a los movimientos que se hagan en la parte superior del cuerpo.
